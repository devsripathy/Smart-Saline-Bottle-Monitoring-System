# Smart Saline Bottle Monitoring System

An IoT-based healthcare solution that continuously monitors IV saline bottle levels and automatically alerts healthcare staff before the bottle becomes empty.

## Overview

Hospitals often require nurses to manually inspect IV bottles, increasing workload and risking delayed replacement. This project automates the monitoring process using sensors and wireless communication.

## Features

- Real-time saline level monitoring
- Automatic low-level detection
- Wireless notification system
- Nurse dashboard
- Reduced manual monitoring
- Low-cost hardware

## Technologies

- ESP32 / Arduino
- Ultrasonic/Load Cell Sensor
- Wi-Fi
- Embedded C++
- IoT
